**Hello fellow developers, welcome to Lucid Evolution!** 

Lucid Evolution, maintained by Anastasia Labs, is an upgraded version of the legacy Lucid library. It features improved APIs, enhanced error handling, better structure, and the latest CML version. The entirety of the library has been created from scratch to address identified areas for improvement
